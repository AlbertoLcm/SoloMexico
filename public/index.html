SoloMéxico

Un sitio web para aprender sobre las culturas de México.

Proposto. Este sitio web es diseñado con el propostito de fomentar la inclusión de las culturas mexicanas en la sociedad actual. Ya que, a lo largo del tiempo, estas tienden a hacer cada vez mas olvidadas provocando como resultado una gran resago social.
Es por esta razón que a continuación se enseñara sobre las culturas mas populares en México
